history Module¶

This module provides some basic helper/formatting utilities, specifically targeted at decoding ReadHistoryData data.

decocare.history.unmask_date(data)[source]¶

Extract date values from a series of bytes. Always returns tuple given a bytearray of at least 5 bytes.

Returns 6-tuple of scalar values year, month, day, hours, minutes, seconds. >>> unmask_date(bytearray( [ 0x6f, 0xd7, 0x08, 0x01, 0x06 ] )) (2006, 7, 1, 8, 23, 47)

>>> unmask_date( bytearray( [ 0x00 ] * 5 ))
(2000, 0, 0, 0, 0, 0)
